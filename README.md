# 🧄 Garlic

**Garlic** is a modern and sleek desktop Git client, built on [Tauri](https://tauri.app), inspired by [GitKraken](https://www.gitkraken.com).

Download for Mac in [releases](https://github.com/rametta/garlic/releases).

- Themes.
- Always free.
- Signed & notarized Mac builds.
- No data collection - all local.
- Bring your own AI key for commit messages.
